# Knowledge Library Index

## When to consult this library
- Before any creative or evaluative task, check foundations/
- Before writing anything user-facing, check craft/ux-writing.md + voices/
- When proposing UI patterns, check taste/ for quality calibration
- When Hannah mentions a specific framework or person, check canon/
